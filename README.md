# UIWindow-Transition
UIWindow transition effect while change window's rootViewController

# Usage

```swift
UIApplication.sharedApplication().keyWindow?.transitRootVc(yourTabBarTwoController(), style: transitionType, duration: nil)
/* there are 11 kinds of transition effects
transitionType:  None, 
                 ZoomOut,
                 ZoomIn,
                 Dissolve,
                 SliceVertical,
                 SliceHorizontal,
                 FlipFromLeft,
                 FlipFromRight,
                 FlipFromTop,
                 FlipFromBottom,
                 CurlUp,
                 CurlDown,
                 Random
  */
```

# Install

- use cocoapods
```swift
pod 'UIWindow-Transition', '~> 0.0.1'
```
- manual import
```swift
just download lib folder and add into your project
```
